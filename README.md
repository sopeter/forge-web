# Forge Web

Marketing website for the [Forge iOS app](https://github.com/sopeter/forge).

## Stack

- Next.js 14+ (App Router)
- Tailwind CSS
- TypeScript
- Deploy: Vercel

## Getting Started

```bash
npx create-next-app@latest . --typescript --tailwind --app --eslint
npm run dev
```

## Design

The full design spec is in [`docs/design-spec.md`](docs/design-spec.md). Use it to guide implementation — it covers page structure, visual style, all six sections, copy, components, and responsive behavior.
